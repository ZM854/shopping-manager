# Shopping Manager

A full-stack web application for creating and managing shopping lists.

## Tech Stack

| Category | Technologies      |
|----------|-------------------|
| Backend  | Go                |
| Frontend | React, TypeScript |
| Database | PostgreSQL        |

## Key Features

- User authentication
- Create and manage shopping lists
- Offline support
- Online synchronization
